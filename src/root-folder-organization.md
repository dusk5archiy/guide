# Root Folder Organization

This is a typical minimal layout of a project:

```txt
project_name/
|-  docs/
|-  src/
|-  tests/
|-  README.md
```

- The top level folder is named after the project.
This folder is referred to as the root folder of the project.
- The `doc` or `docs` folder contains documentations.
- The `src` (source) folder contains the main source code, in other words,
the human-written scripts to define the program's behavior.
- The `test` or `tests` folder holds automated tests to ensure the program's
correctness.
- The `README`, `README.txt` or `README.md` file provides essential
information about the project.

The root folder may include additional folders depending on the 
development tools being used and the nature of the project.
